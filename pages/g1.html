G1: What is User Design?

a. What is User Design?

User Design is the layout and format of an application, website, etc. It helps users like us to use and interact with the application.

b. What is the difference between User Experience Design and User Interface Design? Why does it matter?

The difference between User Experience Design and User Interface Design. That, UX is the information and data that are shown whereas the UI is the overall design and template to put the information and data.
